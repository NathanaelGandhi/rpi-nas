### Setting up software
Make a folder to hold your git clones
```
mkdir -p ~/git
```

Clone this repository
```
git clone https://github.com/NathanaelGandhi/rpi-nas.git ~/git/
```

Run the install script and follow the prompts
```
./install.sh
```
